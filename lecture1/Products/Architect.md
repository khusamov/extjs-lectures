Sencha Architect позволяет вашей команде создавать приложения HTML5 с использованием функций перетаскивания, чтобы вы тратили меньше времени на ручное кодирование, а ваш код приложения оптимизирован для высокой производительности.


Конечный визуальный разработчик приложений для разработки кросс-платформенных приложений HTML5 на настольных и мобильных устройствах.

Ускорение разработки приложений HTML5 с использованием Sencha Architect
Архитектор позволяет вашей команде создавать приложения HTML5 с использованием функций перетаскивания, поэтому вы тратите меньше времени на ручное кодирование, а ваш код приложения оптимизирован для высокой производительности.

Архитектор экономит время разработчиков, удаляя необходимость набирать наиболее шаблонный код. Автоматическое создание кода ускоряет разработку как настольных, так и мобильных приложений. Простое изменение свойств и конфигураций любого компонента внешнего интерфейса JS в окне WYSIWYG устраняет человеческую ошибку, которая часто возникает при ручном кодировании.

Дополнительные компоненты премиум-класса JS - D3, Calendar, Pivot Grid и Exporter - доступны как для современных, так и для классических наборов инструментов. Вы можете легко перетащить премиальные компоненты на панель проектирования Architect и быстро создать сложные веб-приложения.

Архитектор доступен как часть пакетов Ext JS Pro и Premium. Попробуйте Архитектор бесплатно, загрузив пробную версию Ext JS 30 дней. В электронном письме с подтверждением содержится ссылка на Architect, поэтому вы можете просто выбрать его, если вы уже загрузили Ext JS.

Применение
Применение

Архитектор позволяет импортировать темы, созданные с помощью Themer. Индивидуальные темы или модификации, которые вы делаете для предварительно упакованных тем, могут быть применены к вашим приложениям, поэтому они соответствуют внешнему виду бренда вашей организации.

Интеграция Sencha Cmd
Интеграция Sencha Cmd

Архитектор полностью интегрирован с Cmd, что упрощает создание приложений для веб-развертывания или их размещение для собственного развертывания через все основные магазины приложений. Интеграция Cmd автоматизирует многие задачи, связанные с разработкой приложений Ext JS, включая создание производственной сборки и минимизацию кода.
